Project
보헤미안 로스터즈 애플리케이션 기획&디자인 입니다.

보헤미안의 로고를 리디자인 하여 앱에 적용하였으며 기존 브랜드의 앱 환경이 구축되지 않았었기에 타사 브랜드의 레퍼런스를 분석하여 기본에 충실한 UI 환경으로 제작 되었습니다.

브랜드 메인컬러와 상품 이미지를 응요/편집하여 앱에 적용했습니다. 커피의 따뜻함과 차분함을 강조한 브랜드의 특색을 응용하여 전체적인 애플리케이션 컨셉이 반영 하였습니다.

대학교 졸업과제를 통해 보헤미안 카페의 애플리케이션을 기획부터 디자인까지 제작하여 졸업과제로 제출한 작품입니다.
